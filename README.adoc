== Overview

**saur-ssg** is a satellite SSG implementation in the https://github.com/hyperpolymath[hyperpolymath] ecosystem. It provides a standardised https://modelcontextprotocol.io[Model Context Protocol (MCP)] interface to 28 different static site generators, enabling AI assistants to build and manage static websites across diverse technology stacks.

== Features

* **28 SSG Adapters** - Unified interface to static site generators
* **17 Languages** - From Rust to Erlang, Haskell to Julia
* **MCP Compatible** - Standard protocol for AI tool integration
* **RSR Gold Compliant** - Follows Rhodium Standard Repository guidelines
* **Deno Runtime** - Secure, modern JavaScript execution
* **Comprehensive Testing** - Unit, E2E, and security tests

== Quick Start

=== Prerequisites

* https://deno.land[Deno] 1.x or later
* https://github.com/casey/just[just] (optional, for task running)
* Individual SSG toolchains as needed

=== Installation

[source,bash]
----
# Clone the repository
git clone https://github.com/hyperpolymath/saur-ssg.git
cd saur-ssg

# Using Nix (recommended)
nix develop

# Or install tools manually
# See .tool-versions for required versions
----

=== Basic Usage

[source,javascript]
----
// Import an adapter
import * as zola from "./adapters/zola.js";

// Connect to the SSG
await zola.connect();

// List available tools
console.log(zola.tools.map(t => t.name));
// → ["zola_init", "zola_build", "zola_serve", "zola_check", "zola_version"]

// Execute a tool
const result = await zola.tools[0].execute({ directory: "./my-site" });

// Disconnect when done
await zola.disconnect();
----

=== Running Tests

[source,bash]
----
# Run all tests
just test-all

# Or with Deno directly
deno test --allow-read --allow-run tests/
----

== Adapter Interface

Each adapter exports a standard MCP interface:

[source,javascript]
----
// Required exports
export const name = "SSGName";
export const language = "Language";
export const description = "...";

// Connection functions
export async function connect() { }
export async function disconnect() { }
export function isConnected() { return boolean; }

// MCP tools
export const tools = [
  {
    name: "ssg_operation",
    description: "What it does",
    inputSchema: {
      type: "object",
      properties: { /* JSON Schema */ }
    },
    execute: async (params) => { return { result: ... }; }
  }
];
----
